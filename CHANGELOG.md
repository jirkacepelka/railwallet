# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Railgun Vault Wallet
- Dark mode UI with blue accents
- QR code generation for receiving funds
- Transaction history tracking
- Railgun Vault smart contract integration
- Support for Arbitrum One network
- Local transaction storage
- Address validation utilities
- Number formatting utilities
- Custom UI widgets
- GitHub Actions CI/CD pipeline
- Comprehensive documentation

### Features
- Send ETH through Railgun Vault
- Receive ETH with QR code
- View transaction history
- Display real-time ETH/USD prices
- Copy address to clipboard
- Settings screen with network info

### Security
- No private key storage
- Environment variable support for API keys
- Input validation for addresses
- Secure local storage

### Documentation
- README with setup instructions
- Contributing guidelines
- API documentation
- Security warnings
