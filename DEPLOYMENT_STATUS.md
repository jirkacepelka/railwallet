## 📥 How to Get APK

### Option 1: Download from GitHub Releases
1. Go to: https://github.com/jirkacepelka/railwallet/releases
2. Download `app-release.apk` from v1.0.0
3. Install on Android device

### Option 2: Build Locally
```bash
git clone https://github.com/jirkacepelka/railwallet.git
cd railwallet
flutter pub get
flutter build apk --release
# APK: build/app/outputs/apk/release/app-release.apk
```

### Option 3: GitHub Actions Artifacts
1. Go to: https://github.com/jirkacepelka/railwallet/actions
2. Find "Build APK Release" workflow
3. Download artifact from latest run
